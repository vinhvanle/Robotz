Team name

Figuring out a team name that is satisfactory to everyone can be difficult sometimes so we thought “How about naming it nothing at all?”. That’s the reason we came up with the error message – 404 Not Found! is a common error code usually found when browsing the internet, it happens when users tried to access a page that is no longer exist on the host’s server.
